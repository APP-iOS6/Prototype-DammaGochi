# 🥚 주요기능
 _내가 들고다니는 핸드폰 안에서 사는 반려동물을 키우는 듯한 경험을 제공합니다! 마치, 다마고치처럼요 ..🥚_
 <br/>
- **시작 전**: 키우고 싶은 **반려동물 캐릭터**를 생성하세요

<br/>

-**🏠 Home** : 반려동물 여러 마리가 함께 뛰어놀아요. AR 기술로 내 방안에 함께 있는 기분이에요.
- **🛁 Care Zone** : 캐릭터에게 밥도 주고, 놀아주고, 꾸며주며 다마고치처럼 쑥쑥 길러주세요
- **🕹️ Game** : 게임을 하면서 더 재미있게 놀아주세요! 산책도 하고, 친구도 사귀고 공놀이도 하면서 코인도 모아봐요 🪙
- **🏪 Store** : 아이템을 구매하고 반려동물에게 선물해주세요
<br/><br/>

# 🥚 피드백 ✔️
## 시작 전 화면
- 홈 -> 커스텀 버튼 -> 현재 시작 전 화면
- 사진 입력 옵션 추가 X
<br/>

### 홈 화면
- 사용 방법 안내 추가
- 방치형 캐릭터 환경 제작
- ~~사용자의 여러 반려동물 -> 다른 사용자의 반려동물 (커뮤니티형)~~
- 산책시키기 느낌 & 배경 움직이기
<br/>

### 케어존
- My 삭제 및 '카드 공유' 기능 추가
- 프로필에 나이정보 & 설정 추가 (설정 화면 구현)
- 외로움, 포만감 등 정보 확인 및 반려동물 이름 수정
<br/>

### 게임
- 동물 관련 게임으로 (산책, 간식 잘 받아먹기 등)
- 게임 상세페이지 제작 (1 ~ 2개 정도 간략한 구현)
<br/>

### 스토어
- 상품 갯수 추가 & disabled 를 제외 (여러 번 사고 싶은 경우)
